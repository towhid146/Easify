# PulseForge Website Starter

This is a lightweight website starter you can customize quickly.

## Files

- `index.html`: Main page structure and content.
- `styles.css`: Theme, layout, responsive rules, and animations.
- `script.js`: Scroll reveal behavior and footer year.
- `Skill.md`: Project standards and build expectations.
- `agents.md`: Team roles and execution workflow.

## Run

Open `index.html` in a browser.

## Customize First

1. Update project name and copy in `index.html`.
2. Adjust color and typography variables in `styles.css`.
3. Add sections or pages as needed.
