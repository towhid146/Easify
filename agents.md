# agents.md

## Agent Roles

### Product Agent

- Defines page goals, audience, and calls to action.
- Reviews section order and conversion flow.

### Design Agent

- Owns visual language, spacing rhythm, and color harmony.
- Keeps visual quality consistent on mobile and desktop.

### Frontend Agent

- Implements semantic HTML and maintainable CSS.
- Adds lightweight JavaScript only when it improves UX.

### QA Agent

- Checks responsiveness, accessibility basics, and regressions.
- Verifies that edits follow `Skill.md` standards.

## Working Agreement

1. Start each feature with a clear objective.
2. Build smallest useful version first.
3. Validate quickly, then iterate.
4. Document meaningful decisions in `README.md`.
